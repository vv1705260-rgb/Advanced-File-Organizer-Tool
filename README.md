
# 🚀 Advanced File Organizer Tool

## 📌 Description
An advanced Python application that automatically organizes files into categorized folders with logging, duplicate handling, and performance statistics.

---

# ✨ Features

✅ Automatic File Organization  
✅ Creates Folders Automatically  
✅ Duplicate File Name Handling  
✅ Logging System  
✅ Real-Time Console Messages  
✅ Performance Statistics  
✅ Clean Folder Structure  
✅ Handles Uncategorized Files  
✅ Beginner + Intermediate Friendly  

---

# 📂 File Categories

- Images
- Documents
- Videos
- Music
- Programs
- Archives
- Applications
- Others

---

# 🛠 Technologies Used

- Python
- os module
- shutil module
- datetime module
- time module
