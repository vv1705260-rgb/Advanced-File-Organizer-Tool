# 📌 Project Overview

Advanced File Organizer Pro is a professional Python automation project designed to intelligently organize files into categorized folders based on file type. The system automatically detects file extensions, creates folders dynamically, handles duplicate files, generates activity logs, and improves overall digital workspace management.

This project demonstrates real-world Python automation, file system operations, data handling, hashing techniques, and software engineering practices.

---

# ✨ Key Features

## 📂 Automatic File Organization

* Automatically organizes files into categorized folders.
* Detects file extensions intelligently.
* Supports multiple file formats.

## 🧠 Smart Categorization

Organizes files into:

* Images
* Documents
* Videos
* Audio
* Archives
* Python Files
* Web Files
* Others

## 🔁 Duplicate File Detection

* Detects duplicate files using MD5 hashing.
* Moves duplicates into a dedicated folder.
* Prevents unnecessary storage usage.

## 📜 Activity Logging System

* Generates organizer log file.
* Tracks moved files and duplicate detection.
* Records errors and operations.

## ⚡ Professional CLI Interface

* Interactive command-line interface.
* Clean and readable console output.
* User-friendly execution flow.

## 🛡️ Error Handling

* Handles invalid paths gracefully.
* Prevents crashes during file operations.
* Improves reliability and stability.

## 📊 Organization Summary

Displays:

* Total files organized
* Duplicate files found
* Skipped files
* Final completion status

# ▶️ How It Works

1. User enters folder path.
2. Program scans all files.
3. File extensions are detected.
4. Appropriate folders are created automatically.
5. Files are moved into categorized folders.
6. Duplicate files are detected.
7. Log file is generated.
8. Final statistics summary is displayed.
